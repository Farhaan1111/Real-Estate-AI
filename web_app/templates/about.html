About Real Estate AI

Intelligent RERA project assistance powered by advanced AI technology

System Overview

Real Estate AI is an intelligent assistant powered by Hybrid RAG (Retrieval-Augmented Generation) technology, specifically designed to provide detailed information about RERA (Real Estate Regulatory Authority) registered projects across India. Our system combines state-of-the-art AI with comprehensive real estate data to deliver accurate, timely, and actionable insights.

Technology Stack
  • Hybrid RAG Architecture

    Combines semantic and keyword search for optimal retrieval

  • FAISS Vector Database

    Efficient similarity search and clustering

  • BM25 Sparse Retrieval

    Traditional keyword-based search capabilities

  • Sentence Transformers

    State-of-the-art text embeddings

  • OpenRouter LLM Integration

    Advanced language model capabilities

  • Flask Web Framework

    Robust and scalable web application
